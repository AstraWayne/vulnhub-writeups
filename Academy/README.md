# Academy (VulnHub)

**Difficulty:** Medium  
**Focus:** FTP enumeration, file upload abuse, cron-based privilege escalation

This machine involves gaining access through anonymous FTP and credential reuse, followed by exploitation of a file upload vulnerability. Privilege escalation was achieved by abusing a writable script executed by a root-owned cron job.

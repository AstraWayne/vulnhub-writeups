# Dev (VulnHub)

**Difficulty:** Medium  
**Focus:** NFS misconfiguration, web application exploitation, sudo abuse

This machine demonstrates exploitation of a misconfigured NFS share to retrieve sensitive files, followed by web application enumeration and local file inclusion. Root access was obtained by abusing a sudo misconfiguration on the `zip` binary.
